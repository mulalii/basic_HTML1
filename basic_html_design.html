HTML Tags cheat sheet

Headings

Headings from `h1` to `h6` identify blocks and sections.
Every page must have one `h1`

Paragraphs

Paragraphs of text go in `p` tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in `ul` tags. each list item goes in an `li` tag.

Types of lists:

  • Unordered list (`ul`)
  • Ordered lists (`ol`)

Anchors(i.e. links)

Anchor tags create links to other pages. The URL goes in the `href` attribute Go to DuckDuckGo to search for more infomation

Setting `target="_blacnk" will open link in a new window. See MDN for more details.

My HTML Cheat sheet

Anchor tags can also create links to a new location on the same page . Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to

Text Emphasis

Italic

The `em` element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use `strong` element to emphasize text. this shows up as bold text in most browers.
